Drop PNG emoji or sticker assets into this folder.

Two ways to make them appear in the picker:
1. Add a line to ../emoji-catalog.txt
   Format:
   file|id|Display Name|Category|filename.png

2. Or just drop the PNG here.
   The app will auto-discover it and place it in the "Custom" category.

Notes:
- Prefer square PNG files with transparency.
- Keep file names simple, for example: thumbs-up.png
- Reopen the emoji picker to reload the catalog.
